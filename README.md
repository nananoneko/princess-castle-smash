# 🎮 像素公主的城堡大作战 - 游戏说明

## ✅ 已修复的问题

### 1. 画面模糊问题
- **原因**: Canvas 缩放时使用了非整数倍率，导致像素插值模糊
- **解决方案**: 
  - 强制使用整数缩放
  - 添加 CSS `image-rendering: pixelated !important`
  - 禁用字体平滑
- **效果**: 现在像素风格清晰锐利

### 2. 城堡位置问题
- **原因**: 城堡生成 Y 坐标太大，导致方块掉落到公主身上
- **解决方案**:
  - 将生成起点从 Y=80 调整到 Y=20（最顶部）
  - 修改金字塔生成逻辑，从上往下堆叠
  - 确保所有城堡类型都从顶部开始生成
- **效果**: 城堡现在悬挂在屏幕上方，不会碰到公主

### 3. 解压感弱问题
- **原因**: 方块生命值太高，需要多次点击才能摧毁
- **解决方案**:
  - 木块：30 → 10（1 击必碎）
  - 石块：60 → 25（2-3 击）
  - 玻璃：10 → 5（1 击必碎）
  - 金块：100 → 40（4 击左右）
  - 伤害提升：20 → 35
  - 新增冲击波效果（震飞周围方块）
  - 粒子特效：5 → 8 个
- **效果**: 打击感超强，基本一击一个！

## 🎯 游戏特性

### 无限关卡模式
- **第 1-3 关**: 固定金字塔教学关
- **第 4 关**: 混合城堡挑战
- **第 5 关起**: 无限随机生成，层数递增（最多 10 层）
- **难度曲线**: 每 2 关增加一层，最高难度后保持稳定

### 游戏目标
1. 点击屏幕发射爱心弹丸
2. 摧毁所有积木块
3. 获得星级评价（根据准确率）
4. 进入下一关，挑战更高难度

### 操作方式
- **鼠标点击/触摸**: 发射爱心
- **空格键**: 向正上方发射
- **P 键/ESC 键**: 暂停游戏
- **R 键**: 重新开始关卡

## 🔧 技术细节

### 画面清晰度优化
```css
canvas {
    image-rendering: pixelated !important;
    image-rendering: -moz-crisp-edges !important;
    image-rendering: crisp-edges !important;
    -webkit-font-smoothing: none !important;
}
```

### 城堡生成逻辑

## 💡 游戏技巧
1. **优先击破底部支撑块** - 引发连锁倒塌
2. **利用冲击波** - 击中密集区域可震飞多个方块
3. **玻璃块最脆弱** - 优先清除
4. **金块奖励高** - 但需要多次攻击
5. **角度很重要** - 抛物线轨迹需要练习
